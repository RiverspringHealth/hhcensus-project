The majority of changes are minor cosmetic improvements.

The navigation between pages has been moved to the row of buttons at the top to ease moving between pages.

The NOTIFICATIONS page has been modified to add the following features/restrictions

  • A list of all residents is provided and you can pick from that list to fill in the form using
  • The list can be filtered by status and by first letter of last name
  • When a resident is selected, current information in MatrixCare is used to populate the form and that information cannot be edited.
  • It is still possible to enter the resident's name directly in the form for first time admissions; but use the resident selection feature for all other actions to minimize typo's

Issues

The list of residents will occasionally and erroneously display on the screen for a second or two. It is harmless, please ignore it for now.
