This is the new Census Report using data from MatrixCare, including

  • The top row of buttons is used to move between pages.
  • The NOTIFICATIONS page has new features
    • Selecting displays a list of current and prior residents
    • You can quickly filter by the first letter of the last name
    • You can also filter by Status: InHouse, Discharged etc
    • Selecting a resident fills in the form with information from MatrixCare.
  • It is still possible to type directly into the form
  • There are restrictions: such as a discharge will not allow you to enter the room/level of care. It's copied from Matrixcare.

All the data is copied from MatrixCare. If there are errors they need to be fixed in MatrixCare.

  • However, all data is copied from MatrixCare into the Census Report at 11:45pm, changes in MatrixCare will not appear until the next day
